Pre-requisite
-------------
Pycharm editer with robotframework  supported plugins and related libraries installed(seleniumlibrary,JSONLibrary,String,Collections,operating system)

How to execute
-----------------
Go to the path where the project is created and execute below command in the terminal(in my case path is C:\Users\venkas46\PycharmProjects\RobotFramework)

          robot -d Results .\RCB\Rcbplayers.robot

Note:
[.\RCB\Rcbplayers.robot is the path of the robot file in my case  and Results is the folder where logs are stored]
